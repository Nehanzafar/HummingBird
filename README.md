# 🐦 Hummingbird AI using Unity ML-Agents

This project explores reinforcement learning by training a hummingbird agent to navigate a 3D environment and collect nectar from flowers while competing against a human player.

The agent was trained using **Unity ML-Agents** with the **Proximal Policy Optimization (PPO)** algorithm. Throughout training, the agent learned how to fly efficiently, approach flowers, and maximize rewards through trial and error rather than predefined movement logic.

## Features

* Reinforcement Learning with Unity ML-Agents
* PPO-based training
* Physics-based hummingbird movement
* Human vs AI gameplay
* Standalone game build included
* Fully implemented in Unity using C#

## Technologies Used

* Unity
* Unity ML-Agents Toolkit
* Python
* PyTorch
